The `PackageInfo` component displays package dependency information including version changes and change type badges.

## Features

- Version change display (current â†’ new)
- Color-coded change type badges
- Dependencies list
- Description support

## Change Types

| Type | Color | Use Case |
|------|-------|----------|
| `major` | Red | Breaking changes |
| `minor` | Yellow | New features |
| `patch` | Green | Bug fixes |
| `added` | Blue | New dependency |
| `removed` | Gray | Removed dependency |

## Examples

### Basic Usage

```tsx
<PackageInfo
  name="react"
  currentVersion="18.2.0"
  newVersion="19.0.0"
  changeType="major"
/>
```

### With Dependencies

```tsx
<PackageInfo name="react" changeType="added">
  <PackageInfoHeader>
    <PackageInfoName />
    <PackageInfoChangeType />
  </PackageInfoHeader>
  <PackageInfoContent>
    <PackageInfoDependencies>
      <PackageInfoDependency name="react-dom" version="^19.0.0" />
    </PackageInfoDependencies>
  </PackageInfoContent>
</PackageInfo>
```
