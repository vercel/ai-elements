The `GitCommit` component displays git commit details including hash, message, author, timestamp, and changed files.

<Preview path="git-commit" />

## Installation

<ElementsInstaller path="git-commit" />

## Features

- Commit hash display with copy button
- Author avatar with initials
- Relative timestamp formatting
- Collapsible file changes list
- Color-coded file status (added/modified/deleted/renamed)
- Line additions/deletions count

## File Status

| Status | Label | Color |
|--------|-------|-------|
| `added` | A | Green |
| `modified` | M | Yellow |
| `deleted` | D | Red |
| `renamed` | R | Blue |

## Examples

### Basic Usage

```tsx
<GitCommit
  hash="a1b2c3d4e5f6"
  message="feat: Add new feature"
  author="John Doe"
  timestamp={new Date()}
  files={[
    { path: "src/index.ts", status: "added", additions: 50 }
  ]}
/>
```

### Custom Composition

```tsx
<GitCommit {...commit}>
  <GitCommitHeader>
    <GitCommitAuthor />
    <GitCommitMessage />
    <GitCommitActions>
      <GitCommitCopyButton />
    </GitCommitActions>
  </GitCommitHeader>
  <GitCommitContent>
    <GitCommitFiles />
  </GitCommitContent>
</GitCommit>
```
