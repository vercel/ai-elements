The `Agent` component displays a collapsible interface for showing AI agent configuration details. It's designed to represent a configured `ToolLoopAgent` from the AI SDK, showing the agent's model, system instructions, available tools, and output schema.

## Usage with AI SDK

Display an agent's configuration alongside your chat interface.

```tsx title="app/page.tsx"
'use client';

import {
  Agent,
  AgentContent,
  AgentHeader,
  AgentInstructions,
  AgentOutput,
  AgentTool,
  AgentTools,
} from '@/components/ai-elements/agent';

const outputSchema = `z.object({
  sentiment: z.enum(['positive', 'negative', 'neutral']),
  score: z.number(),
  summary: z.string(),
})`;

export default function Page() {
  return (
    <Agent defaultOpen>
      <AgentHeader name="Sentiment Analyzer" model="anthropic/claude-sonnet-4-5" />
      <AgentContent>
        <AgentInstructions>
          Analyze the sentiment of the provided text and return a structured
          analysis with sentiment classification, confidence score, and summary.
        </AgentInstructions>
        <AgentTools>
          <AgentTool name="web_search" description="Search the web" />
          <AgentTool name="read_url" description="Read and parse a URL" />
        </AgentTools>
        <AgentOutput schema={outputSchema} />
      </AgentContent>
    </Agent>
  );
}
```

## Features

- Collapsible interface for showing/hiding agent details
- Model badge in header
- Instructions rendered as markdown
- Tool badges with optional tooltip descriptions
- Output schema display with syntax highlighting
- Composable structure for flexible layouts
- Accessible keyboard navigation
