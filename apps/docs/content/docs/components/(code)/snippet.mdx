The `Snippet` component provides a lightweight way to display inline code snippets with optional syntax highlighting and copy functionality. Simpler than CodeBlock, it's designed for brief code references in text.

<Preview path="snippet" />

## Installation

<ElementsInstaller path="snippet" />

## Features

- Inline mode for embedding in text
- Block mode with syntax highlighting
- Optional copy button
- Compact design for chat/markdown

## Examples

### Inline Code

Use the `inline` prop for embedding code within text.

```tsx
<span>Use the <Snippet code="useState" inline /> hook.</span>
```

### With Syntax Highlighting

Provide a `language` prop for syntax highlighting.

```tsx
<Snippet code="npm install package" language="bash">
  <SnippetCopyButton />
</Snippet>
```

### Without Language

When no language is provided, code renders as plain text.

```tsx
<Snippet code="git clone repo">
  <SnippetCopyButton />
</Snippet>
```
