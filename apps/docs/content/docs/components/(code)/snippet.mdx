The `Snippet` component provides a lightweight way to display inline code snippets with optional syntax highlighting and copy functionality. Simpler than CodeBlock, it's designed for brief code references in text.

<Preview path="snippet" />

## Installation

<ElementsInstaller path="snippet" />

## Features

- Inline mode for embedding in text
- Block mode with syntax highlighting
- Optional copy button
- Compact design for chat/markdown
